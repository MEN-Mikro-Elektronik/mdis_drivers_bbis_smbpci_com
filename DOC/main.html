13BB03 - BBIS Board Driver for SMB controllers

Variants

The following driver variants exist:

Variant

Description

SMBPCI_ICH

for ICH SMBus ctrl. @PCIbus

SMBPCI_SCH

for SCH SMBus ctrl. @PCIbus

SMBPCI_16Z001

for 16Z001 SMBus ctrl. @PCIbus

SMBPCI_16Z001_IO

for 16Z001 SMBus ctrl. @PCIbus - io access

SMBPCI_16Z001_F208

for 16Z001_F208 SMBus ctrl. @PCIbus

SMBPCI_ISA_16Z001

for 16Z001 SMBus ctrl. @ISAbus (driver name: SMBISA_16Z001)

SMBPCI_ISA_16Z001_IO

for 16Z001 SMBus ctrl. @ISAbus - io access (driver name: SMBISA_16Z001)
